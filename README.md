# Irfan's Hammerspoon

Here's a useful collection of shortcuts as well and utilities (light filter) that I use on the daily. I'm going to eventually keep expanding this and start establishing the MenuBar Manager there are tidbits of.

This manager would allow you to efficiently enable and disable shortcuts, as well as customize visuals in order to give the best experience.

## Tips
- Keybind Search
  - [`ctrl` & `option` & `command`] + `[space]`

- Full Screen Window
  - [`shift` & `command`] + `enter`

- Display Switch
  - [`shift` & `command`] + `${n display}` [1-n];
